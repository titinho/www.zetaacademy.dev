Hello, JavaScript!

Nyisd meg kedvenc böngésződ. Ha ez a

  • Chrome, akkor nyomd meg az F12-t és válaszd a Console tabot
  • Firefox, akkor nyomd meg az F12-t és válaszd a Console tabot
  • Safari esetében először engedélyezni kell a Fejlesztői módot:
    • Safari > Settings...
    • Advanced tabon be kell pipálni a Show features for web developers
    • Navigálj el bármelyik oldalra vagy csak gépeld be, hogy about:blank

Ezután a villogó kurzor helyére gépeld be a következő sort:

console.log("Hello, JavaScript!");

Enter-t ütve futtathatod első JavaScript kódodat! A konzolon a következő választ fogod kapni:

Hello, JavaScript!
