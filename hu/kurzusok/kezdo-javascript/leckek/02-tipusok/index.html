Típusok

JavaScript egy gyengén és dinamikusan típusos programozási nyelv.

A JavaScript nyelv típusait két nagy csoportba sorolhatjuk:

  • Egyszerű (primitív) típusok
  • Összetett típusok

Egyszerű (primitív) típusok

A következők az egyszerű típusok

Számok
numerikus adatot tartalmaz, függetlenül attól, hogy az egész vagy lebegő pontos 1, 2.0, 3.14
Szöveges típus (sztringek)
szöveges adatot tartalmaz, például "Hello, ZetaAcademy!"
Bool
ennek a típusnak két értéke lehet true vagy false
Undefined
egy olyan változót jelent, amely deklarálva lett, de értéket még nem kapott undefined
Null
azt fejezi ki, hogy a változó szándékosan nem kapott összetett típusú értéket null
BigInt
olyan egész számértéket tartalmaz, amelyet egy Number típus nem tud biztonságosan kezelni
Szimbólum
egy egyedi és nem módosítható azonosítót jelenít meg

Összetett típusok

Minden összetett típus egy objektum (Object).

Összetett típusok többek között:

  • Egyszerű objektumok
  • Listák: List
  • Halmazok: Set
  • Függvények: function
  • Osztályok: Class
